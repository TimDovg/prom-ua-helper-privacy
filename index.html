Политика конфиденциальности Prom.ua Helper

Дата обновления: 2 июля 2026 г.

Prom.ua Helper помогает продавцам Prom.ua проверять и обновлять истекшие скидки товаров через Telegram-бота и backend-сервис. Расширение Chrome используется как удобная панель управления и подключается к Telegram-профилю пользователя через одноразовый код.

Какие данные мы собираем

Сервис может обрабатывать следующие данные:

  • Telegram user id, chat id, username, имя и фамилию, если они передаются Telegram.
  • Prom.ua API-токен, который пользователь самостоятельно отправляет Telegram-боту.
  • Технические данные подключения расширения: session token, pairing token, код подключения, срок действия кода и id текущей задачи.
  • Данные задач по товарам Prom.ua: id товара, название товара, текущая дата окончания скидки, новая дата окончания скидки, статус обработки и текст ошибки, если она возникла.
  • Технические логи выполнения команд, чтобы пользователь видел результат проверки или обновления.

Как мы используем данные

Данные используются только для основной функции сервиса:

  • подключения расширения Chrome к Telegram-профилю пользователя;
  • проверки наличия сохраненного Prom.ua API-токена;
  • поиска товаров с истекшей скидкой;
  • предпросмотра изменений перед применением;
  • обновления даты окончания скидки после подтверждения пользователя;
  • показа статуса и результата выполнения задачи.

Мы не используем данные для рекламы, продажи, кредитного скоринга, определения платежеспособности или целей, не связанных с работой Prom.ua Helper.

Где хранятся данные

Расширение Chrome хранит в chrome.storage.local только техническое состояние подключения: session token, pairing token, код подключения и id текущей задачи. Prom.ua API-токен, пароли и данные товаров в браузере не сохраняются.

Backend-сервис хранит данные в Supabase. Prom.ua API-токен хранится только на backend в зашифрованном виде и не помещается в исходный код, frontend-код, расширение Chrome или Local Storage сайта.

Передача данных третьим сторонам

Данные передаются только сервисам, необходимым для работы Prom.ua Helper:

  • Telegram используется для общения с ботом и получения команд пользователя.
  • Supabase используется для хранения данных и выполнения backend-функций.
  • Prom.ua Public API используется для получения товаров и обновления скидок по API-токену пользователя.

Мы не продаем пользовательские данные и не передаем их третьим лицам для рекламы или маркетинга.

Доступ к сайтам и браузеру

Расширение не читает историю браузера, не отслеживает клики, движения мыши, нажатия клавиш или прокрутку страниц. Расширение не читает содержимое сайтов и не изменяет страницы Prom.ua.

Разрешение на доступ к хосту Supabase используется только для HTTPS-запросов popup расширения к backend API Prom.ua Helper.

Удаление данных

Пользователь может удалить сохраненный Prom.ua API-токен командой /delete_token в Telegram-боте. После удаления токена сервис больше не сможет выполнять проверки и обновления товаров Prom.ua для этого пользователя, пока пользователь снова не добавит токен.

Пользователь также может отключить расширение от своего Telegram-профиля через интерфейс расширения.

Безопасность

Мы применяем разделение между расширением Chrome и backend-сервисом: секретные данные не хранятся в расширении и не передаются в frontend-код. Prom.ua API-токен хранится на backend в зашифрованном виде.

Контакты

По вопросам конфиденциальности можно обратиться через Telegram-бота @prom_ua_helper_bot.
